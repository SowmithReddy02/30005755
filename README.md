# Library Management System

## Overview
The Library Management System is a Java-based application designed to manage e-books, authors, and user memberships in a library. It provides functionalities to add, view, update, delete, and list e-books, authors, and users.

## Features
- **E-Books Management**: Add, view, update, delete, and list e-books.
- **Authors Management**: Add, view, update, delete, and list authors.
- **User Membership Management**: Register users, view membership details, update memberships, cancel memberships, and list all memberships.

## Prerequisites
- Java Development Kit (JDK) 8 or higher
- MySQL database

## Database Setup
1. Create a MySQL database named `LibraryDB`.
2. Create the following tables in the `LibraryDB` database:Author,EBook,User

This Project can run on any IDE such as : SpringToolSuite(STS),EclipseIDE,etc
